# Web3bridge GIVE-Kenya Program

This repository contains my assignments and practice projects for the Web3bridge GIVE-Kenya Program. It serves as a record of my progress and learning journey throughout the bootcamp.

## Assignments

The repository is organized by folders, each representing a specific milestone or assignment:

*   **[Bills/](./Bills)**: **Assignment 1** - A Command Line Interface (CLI) application built with Rust to manage bills. It demonstrates concepts like structs, HashMaps, interactive menus, and robust user input handling.

More assignments will be added to the root directory as I progress through the program.
